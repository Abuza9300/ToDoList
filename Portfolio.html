MEMORY GAME

A Memory Game is a simple yet engaging game designed to test and improve a player’s memory and concentration. The objective is to find matching pairs of cards or tiles from a set of cards that are flipped face down. Each turn, a player flips over two cards, and if the cards match, they remain face up. If they don’t match, the cards are flipped back face down. The game continues until all pairs are found
